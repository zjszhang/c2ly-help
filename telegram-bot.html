创建 Telegram Bot

Telegram Bot 用于后台登录授权、订单通知、管理通知。每个平台建议使用一个独立 Bot，不要和私人机器人混用。

创建 Bot

1. 打开 Telegram，搜索 @BotFather。
2. 发送 /newbot。
3. 输入 Bot 名称，例如 C2LY Admin。
4. 输入 Bot 用户名，必须以 bot 结尾，例如 c2ly_admin_bot。
5. BotFather 会返回一段 token，例如 123456:ABC...。这就是后台要填写的 Bot token。

安全要求

Bot token 等同于机器人密码，不能发给客户，不能放到公开 GitHub，不能截图发到群里。
如果 token 泄露，立刻在 BotFather 里使用 /revoke 重新生成。

后台配置

1. 打开系统后台 → Réglages modulaires → Sécurité。
2. 填写 Bot token。
3. 填写 Secret webhook。建议使用随机字符串，不要用简单密码。
4. 保存设置。
5. 到 Telegram 管理页面配置管理员和订单通知群。

订单群和管理群

建议分两个群：一个订单通知群，一个管理授权群。
把 Bot 拉进群后，在后台 Telegram 页面刷新群列表，选择用途。
订单群只放厨房/接单人员；管理群只放老板或管理员。

常见问题

搜不到 BotFather：确认搜索的是 @BotFather，不要点广告或相似名字。
后台收不到 Telegram：检查 token 是否完整、Bot 是否被拉进群、管理员是否点击过 Bot 开始按钮。
员工能看到管理通知：说明订单群和管理群没有分开，建议重新设置。
