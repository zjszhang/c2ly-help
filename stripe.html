Stripe 在线支付配置

Stripe 用于客户在线付款。建议先用测试模式确认流程，再切换正式模式。

需要准备

1. Stripe 账号。
2. Publishable key：以 pk_ 开头。
3. Secret key：以 sk_ 开头。
4. Webhook secret：以 whsec_ 开头。

后台填写

1. 打开后台 → 设置 → Paiement Stripe。
2. 测试阶段选择 test，填写测试 key。
3. 正式上线选择 live，填写正式 key。
4. 保存后下一单测试支付。

Webhook

Webhook 地址必须配置正确，否则客户付款成功后，后台可能不能及时更新支付状态。
测试模式和正式模式的 key 不要混用。切换 live 前先确认 Stripe 账号已经完成身份和银行账户验证。
