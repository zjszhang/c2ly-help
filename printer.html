自动打印机配置

系统当前主要支持 Star CloudPRNT 自动打印。购买打印机前必须确认型号兼容。

推荐型号

品牌型号建议
StarmC-Print3 CloudPRNT优先推荐，用于厨房订单打印。
StarTSP100IV / TSP143IV CloudPRNT可以使用，购买前确认支持 CloudPRNT。

不推荐

普通 USB 小票机不适合本系统自动打印。网页后台不能直接稳定控制普通 USB 打印机。
只写着“热敏打印机”但不支持 CloudPRNT 的型号，不要购买。

后台设置

1. 打开餐馆后台，进入 Réglages modulaires。
2. 进入 Commandes，找到 Imprimante Star。
3. 设置 Impression active = Oui。
4. Media type 建议保持 Auto。
5. 保存设置。

打印机配置

1. 打开后台 Imprimante 页面。
2. 复制后台显示的 Server URL。
3. 进入打印机的管理页面，找到 CloudPRNT / Server URL。
4. 粘贴 Server URL 并保存。
5. 等待 1 到 2 分钟，后台应显示打印机在线。
6. 点击测试打印，确认厨房能收到测试票据。

排查

后台没有显示打印机：检查打印机是否联网，CloudPRNT Server URL 是否正确。
有订单但不打印：检查后台打印开关、订单支付状态、打印任务是否失败。
打印乱码或格式不对：Media type 改回 Auto，并重新测试。
